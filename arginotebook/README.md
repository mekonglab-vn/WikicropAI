# WikiNotebook

> **Một phần mở rộng Sổ Tay Nông Nghiệp của Wikicrop**

---

## 1. Giới thiệu

Đây là repo về **WikiNotebook**, một phần mở rộng Sổ Tay Nông Nghiệp của Wikicrop.

### Các chức năng cơ bản

- Trích xuất đa dạng loại tài liệu khác nhau và lưu lại vị trí trích xuất
- Tạo bài viết có kèm theo vị trí thông tin được sử dụng
- Tạo sơ đồ tư duy từ bài viết
- Tải bài viết đã cho để admin có thể nhìn thấy
- Quản lý nguồn, quản lý dàn ý

---

## 2. Cấu trúc

- **MyNotebook**: thư mục extension frontend dùng để copy vào thư mục Wikicrop
- **arginotebook_backend**: thư mục backend FastAPI cho hệ thống
- **HuongDanCaiDat_WikiNotebook.pdf**: file ghi thông tin và hướng dẫn cài đặt

---

## 3. Hướng dẫn cài đặt

```bash
git clone https://github.com/HoangPhuc-edward/WikiNotebook.git
```

Hãy làm theo hướng dẫn trong file **HuongDanCaiDat_WikiNotebook.pdf** đính kèm.
